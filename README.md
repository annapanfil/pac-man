Written just for fun.

## Technologies

## Setup

You need to have [python] and pip installed

$pip3 install -r requirements.txt

**$python3 -m pac_man**

On Linux you can just run `pac_man.sh` (If access denied, try `chmod u+x pac_man.sh`)
